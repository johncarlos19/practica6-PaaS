T-Shirt store is a versatile Bootstrap 4 e-commerce template. It's great for selling a single product or small product range. 

It contains 3 most important pages that you might need: homepage (with a range of useful elements), text page and a shopping cart page with a payment form.

This template comes already with 6 colour variants but also with the SASS source files to make the customization of the template even easier and more flexible, just as you need it.

Bootstrap 3 version of this template (with LESS sources) is also part of the download packaage.

This e-commerce theme comes with many cool elements and features including:

  • Hero section to briefly introduce the product and catch user's attention
  • CarroCompra.Main and distinctive features section
  • Classy icon fonts, a special selection of e-commerce icons 
  • Gallery section with lightbox
  • Forms with JavaScript validation
  • Products slider, new arrivals
  • Popup window for more details on products
  • Testimonials carousel
  • Valid Html5 and CSS3 markup
  • Fully responsive
  • Well-structured code
  • SEO friendly
  • Cross-browser compatibility (Chrome, Safari, Firefox, IE 9 +, Opera)
  • 6 colour variants
  • and much more
